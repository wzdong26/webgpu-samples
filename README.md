# WebGPU Samples

## Introduction

Recently chrome released WebGPU and WebGPU is now available by default in chrome 113. This repository is used to help learn WebGPU.

## Install and Run

-   `pnpm i` or `npm i`
-   `pnpm dev` or `npm run dev`
-   Local: [http://localhost:8081/](http://localhost:8081/)
